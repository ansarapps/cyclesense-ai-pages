CycleSense AI - Privacy Policy

Your privacy is important to us.

Data Collection

CycleSense AI collects user data such as email (for login), and health-related data (period tracking, symptoms) only when users choose to use specific features.

Data Storage

Data is primarily stored locally on the user's device. Optional features include:

  • Google Drive backup (user-controlled)
  • Partner mode (secure encrypted storage)

Data Usage

Data is used only to provide app functionality and is not sold to third parties.

Data Security

All data is encrypted in transit using secure protocols.

Account & Data Deletion

You can request deletion of your data by:

  • Using the in-app delete option
  • Sending an email to cyclesenseai@gmail.com

All data will be deleted within 7 days of request.
